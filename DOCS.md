## ⚙️ Функционал

### Воспроизведение
| Функция | Описание |
|---------|----------|
| ▶️ Play/Pause | Воспроизведение/пауза |
| ⏭ Next/Previous | Следующий/предыдущий трек |
| 🔀 Shuffle | Случайный порядок |
| 🔁 Repeat | Повтор (выкл/все/один) |
| ⏪ Seek | Перемотка прогресс-бара |
| 🔊 Volume | Регулировка громкости |
| ⚡ Speed | Скорость (0.5x - 2.0x) |

### Библиотека
| Функция | Описание |
|---------|----------|
| 📤 Import | Загрузка MP3/FLAC/WAV/M4A |
| ❤️ Favorites | Избранные треки |
| 📁 Playlists | Создание плейлистов |
| 🔍 Search | Поиск по библиотеке |
| 🗑 Delete | Удаление треков |
| 📊 Filters | Все/Любимые/Офлайн |

### Интерфейс
| Функция | Описание |
|---------|----------|
| 🎨 Dark Theme | Тёмная тема (как Dotify) |
| 🔴 Red Accent | Красные акценты |
| 📱 Bottom Tabs | Нижняя навигация (4 вкладки) |
| 🎵 Mini Player | Мини-плеер внизу |
| 📱 Full Player | Полный плеер на весь экран |
| 🎤 Lyrics | Текст песни (готово к интеграции) |

## 🔧 Технические детали

### Системные требования
- **iOS:** 16.0+
- **Swift:** 5.9+
- **Xcode:** 15.0+
- **Размер:** ~5 MB (без музыки)

### Разрешения
```xml
NSAppleMusicUsageDescription - Доступ к медиатеке
NSPhotoLibraryUsageDescription - Доступ к фото
UIBackgroundModes/audio - Фоновое аудио
UIFileSharingEnabled - Общий доступ к файлам
```

### Поддерживаемые форматы
- ✅ MP3
- ✅ FLAC
- ✅ WAV
- ✅ M4A (AAC)
- ✅ OGG
- ✅ AIFF

### Хранение данных
```swift
UserDefaults:
- fmp_tracks       // Список треков (JSON)
- fmp_playlists    // Плейлисты (JSON)
- fmp_favorites    // ID избранных треков (Set)
- fmp_settings     // Настройки приложения (JSON)
```

## 🚀 Компиляция

### Локально (нужен Mac)
```bash
cd FreeMusicPlayer-iOS
xcodebuild -project FreeMusicPlayer.xcodeproj \
  -scheme FreeMusicPlayer \
  -configuration Release \
  -sdk iphoneos \
  CODE_SIGNING_ALLOWED=NO \
  CODE_SIGNING_REQUIRED=NO
```

### Через GitHub (Windows/любая ОС)
1. Загрузи проект на GitHub
2. GitHub Actions автоматически соберёт IPA
3. Скачай из артефактов
4. Установи через Sideloadly/AltStore

**Подробная инструкция:** см. `DEPLOY.md`
